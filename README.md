# Project smulberohub

My personal homelab focused on infrastructure, networking and security learning. <br>
This repository is used to document my homelab setup with reproducibility in mind.

## Goals
- Use Hypervisor type 1 as base
- Deploy DNS server with high availability
- Service discoverability via Local DNS
- Deploy media server
- Deploy dashboard service
- Secure remote access

## Technologies
- Proxmox VE
- Pi-hole
- Nginx Proxy Manager
- Jellyfin
- Glance
- Wireguard

## Other goals
- Functional IP Grouping
- Use containers
